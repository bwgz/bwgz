Quotation

Quotation is comprised of three applications which provide access to quotations, authors, and subjects. Each application supports a common feature set:

  • Provide a random quotation.
  • Quotations by author and subject
  • Search for a quotation by keyword, author, or subject.
  • Share a quotation using popular social media providers.

The Quotation applications are:

  • quotation.bwgz.org — Web application written using AngularJS and bootstrap.
  • Quotation for Google Chrome™ — .
