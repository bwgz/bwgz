CA Portal

Overview
CA Portal is an application framework for creating Composite Dashboard applications. These applications combine the elements of a business activity in a single coordinated application and user interface. CA Portal steamlines the personalized delivery of complex information to your business and IT users in the context of their business processes. It is designed to be easy to implement and optimize the distribution of information assets across your enterprise. Leveraging common industry standards for reliability and extensibility, CA Portal is designed to facilitate integration and aggregation of information into a personalized, intelligent, and engaging Web-based environment enabling collaboration, business insight and effective decision-making.
Top Three Key Features
  • Open, Configurable and Customizable Deployment Based on Open Standards
  • Effective Content Delivery for Business Activity Monitoring (BAM) and Business Process Management (BPM)
  • Externalized Identity and Access Management
How it works
  • The user sees a Web portal that combines the various applications needed to complete a business process.
  • Various views combines performance metrics with knowledge management, enterprise search, and online collaboration.
  • The user can to respond to events as they emerge across the business.
Architecture, Standards, Best Practices
  • Tiered web framework - A J2EE application providing performance, scalability, reliability, availability, maintainability, manageability, security.
  • J2EE application standards - JAAS, JSP, EJB, JCA, RMI, JCR
  • User management and security interface - Java Authentication and Authorization Service (JAAS) for interoperability and extensibility. Embedded user management and security packaged is included.
  • Suports portlet standards - Portable Java Portlets (JSR-168), Remote Portlets (WSRP).
  • Embedded content management system for standard access to documents and content. Supports Java Content Repository API (JSR-170), WebDAV.
  • Embedded workflow engine for automating business processes such as content management approvals and shared page distribution.
